# Cognitus Pay Role Matrix

## Staff Member

Can:

- View own payroll profile
- View own paystubs
- Submit pay requests
- Submit raise requests
- Submit adjustment requests

## Finance Employee

Can:

- View finance records
- Maintain revenue and expense logs
- Prepare payroll
- Create paystubs
- View pay requests

## FC Supervisor

Can:

- Do Finance Employee tasks
- Adjust payroll
- Correct paystubs
- Apply bonuses
- Apply deductions and holds
- Review payroll issues

## Chief Financial Officer

Can:

- Do FC Supervisor tasks
- Manage pay rates
- Lock payroll periods
- Finalize payroll periods
- Override finance corrections

## Management

Can:

- Review managed employees
- Submit management audits
- Flag payroll or performance concerns

## Board of Directors

Can:

- View finance records
- Edit logs
- Soft-delete logs
- Review audit history

## Executive

Can:

- Approve pay requests
- Deny pay requests
- Approve raises
- Deny raises
- Edit or delete logs
- Manage users if granted

## Owner

Can do everything.

Owner-only actions:

- Create staff accounts
- Reset PINs
- Disable staff access
- Change roles
- Protect bootstrap account
